# oewang — Native App (Flutter)

Mobile application for oewang, built with Flutter.

## Setup

Flutter SDK is required. Install it via the official guide or with [`fvm`](https://fvm.app) for version management:

```bash
# Option A — Official installer (one-time)
brew install --cask flutter

# Option B — fvm (recommended for version pinning)
brew tap leoafarias/fvm
brew install fvm
fvm install stable
fvm use stable
```

Then scaffold the Flutter project **inside this directory**:

```bash
cd apps/native
flutter create --org com.oewang --platforms android,ios .
```

> **Why `.` at the end?** It scaffolds into the current directory instead of creating a nested sub-folder.

## Running

```bash
# From apps/native/
flutter run              # run on connected device / emulator
flutter test             # run unit tests
flutter build apk        # release APK (Android)
flutter build ios        # release iOS build
```

## Monorepo Notes

- This directory has **no `package.json`** intentionally — bun and Turborepo will ignore it completely.
- All Flutter dependencies are declared in `pubspec.yaml` (generated by `flutter create`).
- Root `.gitignore` already covers Flutter-generated artifacts (`build/`, `.dart_tool/`, `ios/Pods/`, etc.).
- This app consumes the `apps/api` REST API — use the `NEXT_PUBLIC_API_URL` value (found in the root `.env`) as your base URL.
